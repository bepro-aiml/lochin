# Module 2 — Final Lab Scenarios

## Today's Task (No Coding)

You are NOT writing Python today. You are **defining the problem** and **justifying your solution approach** in writing.

### Step-by-step

1. **Read all 10 scenarios below** as a team.
2. **Pick ONE.** Open its `problem_statement.md` for the dataset spec, hygiene rules, and deliverable.
3. **Download the dataset** and clean it according to the platform's hygiene rules (lowercase columns, ISO timestamps, explicit `NaN`). Save the cleaned file — you will use it in implementation week.
4. **Fill out `documentation_phase_1_what.md`** — describe what tools, libraries, methods, and stages you propose to use.
5. **Fill out `documentation_phase_2_why.md`** — justify *why* each of those tools and methods is the right fit for the chosen scenario.
6. **Push both completed docs** to your group repo under your team's submissions folder.

The mentor will not approve any code until **both phase docs are signed off**.
